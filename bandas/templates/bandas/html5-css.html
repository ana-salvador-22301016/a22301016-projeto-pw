Seletores CSS

O CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para descrever a apresentação de um documento HTML. Ela define como os elementos HTML devem ser exibidos na tela, no papel ou em outras mídias.

Seletores CSS Utilizado Comentário
Seletor de elemento (tag) done Seleciona elementos HTML pelo nome da tag. Utilizado para estilizar todas as instâncias de uma tag específica. Exemplo
Seletor de classe (.) done Seleciona elementos HTML pela classe. Utilizado para estilizar elementos com a mesma classe. Exemplo
Seletor de ID (#) done Seleciona um único elemento HTML pelo ID. Utilizado para estilizar elementos únicos. Exemplo

Combinadores de seletores

Combinadores Utilizado Comentário
Espaço ( ) done Seleciona elementos que são filhos de outro elemento. Utilizado para estilizar elementos específicos dentro de um contêiner. Exemplo
Combinador filho (>) done Seleciona elementos que são filhos diretos de outro elemento. Utilizado para estilizar elementos imediatamente dentro de um contêiner. Exemplo
Combinador de irmão adjacente (+) done Seleciona o próximo elemento irmão diretamente após o primeiro elemento. Utilizado para estilizar um elemento que segue imediatamente outro elemento. Exemplo
